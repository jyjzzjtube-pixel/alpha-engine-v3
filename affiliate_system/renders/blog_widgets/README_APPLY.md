# 블로그 위젯 에셋 적용 가이드

## 생성된 에셋
- `widget_consultation.jpg` - 무료 상담 신청 CTA (메인)
- `widget_portfolio.jpg` - 성공 포트폴리오
- `widget_about.jpg` - BRIDGE ONE 소개
- `widget_trend.jpg` - 2025 창업 트렌드
- `widget_contact.jpg` - 연락처 안내
- `blog_header_banner.jpg` - 블로그 타이틀 배너 (966x200)
- `blog_profile_logo.jpg` - 프로필 로고 (200x200)

## 적용 방법
1. Chrome을 디버깅 모드로 열기: `chrome --remote-debugging-port=9222`
2. blog.naver.com/jyjzzj 에 로그인
3. 관리자 > 꾸미기 > 레이아웃/위젯 설정
4. 외부 위젯 등록에서 이미지 위젯 추가
5. 각 위젯 이미지를 업로드하고 링크 연결

## 브랜드 컬러
- 네이비: #1a1a2e
- 레드 액센트: #e94560
- 화이트: #ffffff
